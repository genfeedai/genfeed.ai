# Vision — Genfeed.ai

## Long-Term Direction

Genfeed.ai aims to be the open-source operating system for AI-powered content operations — where individuals and teams configure autonomous agents that continuously generate, optimize, and distribute content across every relevant platform, with minimal human intervention beyond strategic oversight. Self-hosted, extensible, and community-driven.

## Strategic Priorities

### Near-Term (Stabilization)
1. Stabilize the consolidated monorepo after cloud + core migration
2. Harden self-hosted deployment with Docker and clear documentation
3. Prove autonomous agent-driven content workflows end-to-end
4. Maintain CI/CD quality gates and test coverage

### Medium-Term
1. Grow the open-source community and contributor base
2. Expand GPU pipeline capabilities (images, videos, voices)
3. Deepen analytics and content performance feedback loops
4. Grow integration ecosystem beyond current 48+ platforms
5. Polish desktop and mobile app experiences

### Long-Term
1. Fully autonomous content operations requiring only strategic configuration
2. AI-driven content optimization based on performance data
3. Community marketplace for content templates, workflows, and agent configurations
4. Enterprise-grade team operations via `ee/` with advanced permissions and audit trails

## Open-Source Model

- **Core (AGPL-3.0)**: Full self-hosted deployment, BYOK execution, all content features
- **Enterprise (`ee/`)**: Multi-tenancy, SSO, billing, analytics, team collaboration (commercial license)
- Community contributions welcome — integrations, workflow nodes, agent types

## Design Philosophy

- Precise, premium, and trustworthy
- Operational confidence over flashy aesthetics
- Calm neutral surfaces with strong information hierarchy
- One visual interaction pattern per job
- Explicit states over ambiguous ones
