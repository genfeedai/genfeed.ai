**Rule:** Write durable memory as active target-state guidance: name the current source of truth, direct URL, command, or desired behavior first.

**Why:** Memory text primes future agents. The safest durable form is the current target state plus the action to take.

**How to apply:**
- Prefer `Use X at URL Y` over historical comparisons.
- Keep routine summaries focused on active sources and current workflows.
- Include history only when it is required to execute the current task safely.
