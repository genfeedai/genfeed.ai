Genfeed.ai is a TypeScript-first open source AI OS for content creation. Monorepo uses Turborepo + Bun workspaces.

**Why:** Understanding the workspace layout is essential for scoping changes correctly.
**How to apply:** Always scope changes to specific apps/packages. Use `@genfeedai/*` path aliases. Follow existing patterns.
