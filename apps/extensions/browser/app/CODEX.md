# Browser Extension

Codex-specific entry point. Plasmo-based Chrome/Firefox extension for Genfeed.ai.

## Codex-Specific Notes

- **No browser**: cannot test extension in sandbox. Focus on React components and logic.
- **Plasmo framework**: builds to Chrome/Firefox extensions.

## Key Entry Points

- Extension popup and content scripts in `src/`
- Shared types from parent `packages/`
