# Browser Extension Project

**Plasmo-based Chrome/Firefox extension.**

Rules auto-load from `.claude/rules/` (symlinked to `.agents/rules/`). Docs in `../../../../.agents/`.

## Tech Stack

- Plasmo framework
- React + TypeScript
- Tailwind CSS

## Commands

```bash
bun run dev     # Development mode
bun run build   # Production build
```
