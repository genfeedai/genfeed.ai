# Mobile App

Codex-specific entry point. React Native app with Expo for Genfeed.ai.

## Codex-Specific Notes

- **No simulator**: cannot run iOS/Android in sandbox. Focus on component logic.
- **Expo Router**: file-based routing in `app/` directory.
- **NativeWind**: Tailwind CSS for React Native.

## Key Entry Points

- Routes in `app/` (Expo Router)
- Shared types from parent `packages/`
