# Contributing

The canonical contribution guide is
[`CONTRIBUTING.md`](https://github.com/genfeedai/genfeed.ai/blob/master/CONTRIBUTING.md).
It documents the supported Bun toolchain, local setup, focused verification,
repository boundaries, and pull-request process.

Report suspected vulnerabilities through
[`SECURITY.md`](https://github.com/genfeedai/genfeed.ai/security/policy), not a
public issue.
