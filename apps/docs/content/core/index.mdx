# Genfeed Core (Free & Open Source)

The workflow engine for AI content generation. Self-hosted, bring your own API keys.

## How It Works

```
Your API Keys → Workflow Engine → AI Content
(Replicate, OpenAI, etc.)
```

1. **Connect your API keys** — Replicate, OpenAI, Anthropic, ElevenLabs
2. **Build workflows** — Visual drag-and-drop pipeline builder
3. **Generate content** — Text, images, video, audio
4. **Export anywhere** — Local files, S3, webhooks

## Why Core?

|              | Core (Free)           | Cloud                 |
| ------------ | --------------------- | --------------------- |
| **Cost**     | Free + API costs      | $299/mo               |
| **Data**     | Your servers          | Our servers           |
| **Limits**   | None                  | Plan-based            |

Core can also call selected managed Cloud services when you explicitly configure a Cloud API key. For the exact local, BYOK, managed, and Cloud split, see [Execution Boundaries](/core/execution-boundaries).

## Quick Start

```bash
git clone https://github.com/genfeedai/genfeed.ai.git
cd genfeed.ai
bun install
cp .env.example .env.local
bun run env:sync local
# Add your API keys to the root .env.local
bun dev
```

Open `http://localhost:3000` and start building workflows.
