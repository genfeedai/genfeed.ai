# Advanced Surfaces

Genfeed's source tree includes workflow orchestration, webhooks, realtime
events, publishing integrations, analytics modules, agent tools, and extension
clients. Availability depends on deployment mode, provider configuration, and
commercial entitlement.

## Verify before relying on a surface

1. Confirm the module or client exists in the current release.
2. Confirm the route or workflow node is registered for your deployment mode.
3. Confirm required provider credentials and approvals.
4. Confirm whether execution is local/BYOK or an explicit managed Cloud call.
5. Run a focused end-to-end check for the path you intend to operate.

## Boundaries

- Community is a single-tenant web/API/MCP Docker distribution.
- Cloud owns managed operations, multi-tenant governance, and managed
  publishing/analytics surfaces.
- The current `ee/` tree contains billing and harness packages; broader managed
  inference infrastructure is private and is not shipped here.
- Desktop is macOS-first. Mobile and extension workspaces do not imply public
  store listings.

No SOC 2, GDPR, encryption, SSO/SAML, support, or SLA claim should be inferred
from the presence of source code. Use formal commercial and security materials
for any such requirement.
