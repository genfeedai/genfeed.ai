# Studio

Studio is the product workspace for prompt-driven generation. It combines model
selection, output settings, references, and supported brand context with the
configured generation providers.

Generated assets can appear in product library surfaces when the active flow
persists them. Available content types, models, dimensions, and actions depend
on the provider registry and deployed release.

These docs intentionally avoid undocumented keyboard shortcuts and do not claim
that every generated asset is automatically saved or publishable.

See [Available Models](/models) and [Brands](/cloud/brands).
