# Billing and Credits

Cloud billing, subscriptions, credit packs, and managed-inference usage are
commercial surfaces. Fixed prices and included credits are intentionally not
copied into these docs because they can change independently of a repository
release.

Use the [current pricing page](https://genfeed.ai/pricing) and the checkout UI as
the source of truth before purchasing.

## Repository boundary

- The public API contains generic credit and subscription contracts plus OSS
  stubs used by Community.
- The current `ee/` tree contains the commercial billing-provider package.
- The Community image excludes `ee/` and does not sell managed credits locally.
- A Community or desktop client can use managed inference only through an
  explicit Cloud API key and Cloud-credit authorization.

This page does not promise a trial, fixed price, included quota, seat count,
support tier, or SLA.
