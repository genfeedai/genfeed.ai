# Genfeed Cloud

Genfeed Cloud is the hosted deployment of the web product and server stack. The
repository contains its application source and deployment automation; live
availability, plan terms, quotas, and support channels are operational concerns
outside the source distribution.

## Cloud boundary

Cloud can add:

- managed infrastructure and managed inference;
- multiple isolated organizations;
- hosted publishing and scheduling operations;
- managed analytics; and
- team and commercial governance.

Exact capabilities depend on the deployed release and active entitlement. Use
the live product for current account behavior, and see
[Execution Boundaries](/core/execution-boundaries) for what remains local in
Community.

## Start

Open [app.genfeed.ai](https://app.genfeed.ai). Account creation, plan selection,
and onboarding can change independently of these docs.
