# Genfeed Cloud

Managed SaaS platform for teams who want one control plane for generation, publishing, analytics, and automation.

## Last Verified

- **Date:** 2026-04-07
- **Implemented state source:** `apps/` codebase
- **Delivery state source:** GitHub issues + Mission Control project (#11)

## What Cloud Adds

| Area | Cloud Capability |
|---|---|
| Organization model | Multi-tenant workspaces |
| Brand operations | Brand profiles and content context |
| Team operations | Member and access workflows |
| Publishing | Multi-platform workflows |
| Management UI | Admin panel, studio app, marketing website |

## Quick Start

1. Create account and organization
2. Configure brand context
3. Land in Overview inside the app
4. Move through the loop: trends -> create -> review -> publish -> analytics -> automations
