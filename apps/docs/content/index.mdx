# Genfeed.ai

**Open-source AI OS for content creation.**

Genfeed combines a visual workflow system, provider-backed generation, content
management, and programmatic interfaces in one monorepo.

## Ways to use Genfeed

| Surface        | Distribution status                                                                                                 |
| -------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Community**  | Single-tenant self-hosted web/API/MCP stack distributed through Docker.                                             |
| **Cloud**      | Hosted SaaS operated by Genfeed; availability, plans, and managed services are outside the repository distribution. |
| **Desktop**    | macOS-first Electron source and release workflow. Tagged installers appear in GitHub Releases when published.       |
| **Mobile**     | Expo/React Native source; its EAS config is disabled/not actively developed, and no app-store listing is linked.    |
| **Extensions** | Chrome and VS Code extension source. Build/package workflows exist, but no public store listing is claimed here.    |

Community uses the repository's default
[AGPL-3.0-or-later license](https://github.com/genfeedai/genfeed.ai/blob/master/LICENSE),
except where a package carries its own license notice. Code under `ee/` uses its
separate commercial license.
