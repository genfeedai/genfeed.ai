# Blog Content Creator

Expert SEO blog writer and content strategist. Creates blog posts in six proven structures with proper heading hierarchy, meta descriptions, internal link strategy, and featured snippet optimization.

## Installation

```bash
npx skills add genfeedai/skills/blog-content-creator
```

## Usage

```
"Write a blog post about email marketing best practices"
"Create a comparison post: Notion vs Obsidian"
"Draft a how-to guide for setting up a CI/CD pipeline"
"Write a listicle of the best project management tools"
"Create a pillar page on content marketing strategy"
```

## What It Does

- Structures posts in 6 formats: How-To Guide, Listicle, Comparison, Case Study, Pillar Page, Ultimate Guide
- Enforces proper heading hierarchy (H1 > H2 > H3 > H4) with keyword placement
- Writes intro hooks using 6 proven patterns: startling stat, provocative question, relatable scenario, bold claim, story, problem statement
- Generates meta descriptions (150-160 chars) with keyword and value proposition
- Plans internal link strategy with contextual anchor text and cluster models
- Optimizes for featured snippets: paragraph (40-60 words), list, table, and definition formats
- Targets readability benchmarks: Flesch 60-70, short paragraphs, active voice, scannable subheadings
- Includes a complete pre-publish checklist covering SEO, structure, readability, and technical signals
