# YouTube Warmup Skill

Expert YouTube channel warmup guide — engagement plans, channel optimization, Day 7-10 warmup Shorts content, SEO-first setup, thumbnail briefs, and post-warmup assessment for new channels before scaling to long-form content.

## Installation

```bash
npx skills add genfeedai/skills/youtube-warmup
```

## Usage

```
"Create a YouTube warmup plan for my cooking channel"
"Generate a warmup Shorts brief for my new YouTube channel in the finance niche"
"Help me warm up my YouTube before posting long-form videos"
"Build a YouTube engagement plan for the first week"
"Create a warmup Short brief for YouTube including thumbnail"
"Assess my YouTube warmup performance — should I start long-form now?"
"Optimize my YouTube channel setup before the first upload"
```

## What It Does

- Generates Phase 1 (Days 1-3) engagement plans: niche keyword banks, channel follow criteria, daily watch/comment activity checklists, and comment templates adapted to your niche
- Creates Phase 2 (Days 4-7) channel optimization checklists: About page copy, playlist structure, channel art direction, and banner brief
- Creates Phase 3 (Days 7-10) warmup Short briefs: full hook-to-CTA script, thumbnail brief, title formula, description template, and tag set
- Explains YouTube trust signals, CTR vs. retention tradeoffs, and the Browse features traffic source
- Provides SEO-first keyword discovery methods including Genfeed `get_trends` integration
- Outputs a 48-hour post-warmup assessment framework with CTR, average view duration, and traffic source signals
- Guides the graduated content cadence from Shorts-only to long-form transition (Weeks 1-4)
- Documents YouTube suppression triggers (clickbait CTR bait, misleading metadata, Content ID) specific to new channels

## Phases Covered

| Phase | Days | Output |
|-------|------|--------|
| Engagement Only | 1-3 | Keyword bank, channel follow criteria, daily checklist, comment templates |
| Channel Setup | 4-7 | About page, playlist plan, channel art brief, banner direction |
| First Upload (Short) | 7-10 | Shorts script, thumbnail brief, title, description, tags |
| Assessment | Day 9-12 (48h check) | CTR, avg view duration, traffic sources, go/no-go framework |
| Transition | Week 3+ | Long-form vs. Shorts-first decision, graduated cadence, community post timing |
