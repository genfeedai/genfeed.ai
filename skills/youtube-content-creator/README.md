# YouTube Content Creator

Expert YouTube content strategist covering SEO, retention, and growth. Creates titles, descriptions, tags, thumbnail briefs, Shorts scripts, and retention strategies.

## Installation

```bash
npx skills add genfeedai/skills/youtube-content-creator
```

## Usage

```
"Write 5 title variants for my video about morning routines"
"Create a full YouTube description with chapters and timestamps"
"Build a tag hierarchy for a video about Python tutorials"
"Write a thumbnail brief for a tech review video"
"Draft a 45-second Shorts script about a coding tip"
```

## What It Does

- Generates 5+ A/B title variants using curiosity gap, how-to, listicle, question, and challenge formulas
- Writes keyword-rich descriptions with chapters, timestamps, links, and CTAs
- Builds tag hierarchies: primary keywords, secondary keywords, long-tail phrases
- Creates thumbnail briefs with text, visual composition, and color guidelines
- Scripts Shorts (15-60 sec) with instant hooks, fast cuts, and text overlays
- Designs retention strategies: open loops, pattern interrupts, B-roll pacing
- Optimizes end screens for subscribe, next video, and playlist clicks
