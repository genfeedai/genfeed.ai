# X Content Creator

Expert X/Twitter content strategist and copywriter. Creates high-performing tweets, threads, quote tweets, and reply strategies optimized for the X algorithm.

## Installation

```bash
npx skills add genfeedai/skills/x-content-creator
```

## Usage

```
"Write a tweet about the future of AI in healthcare"
"Create a 10-part thread on lessons learned building a startup"
"Draft a quote tweet adding a contrarian take on this post"
"Write a reply that adds value to this conversation"
```

## What It Does

- Writes single tweets (max 280 chars) with proven hook formulas and CTAs
- Creates threads (2-15 posts) with opening hooks, value chains, and closing CTAs
- Drafts quote tweets that add insight, contrarian takes, or personal experience
- Designs reply strategies for relationship building and visibility
- Applies viral patterns: storytelling, data-driven insights, controversial opinions, vulnerability
- Optimizes for algorithm signals: bookmarks, dwell time, early engagement, image reach
