# Instagram Warmup Skill

Expert Instagram account warmup guide — 14-day engagement plans, Stories trust-building strategy, first Reel warmup content briefs, carousel formats, and post-warmup assessment for new and purchased accounts before posting promotional content.

## Installation

```bash
npx skills add genfeedai/skills/instagram-warmup
```

## Usage

```
"Create an Instagram warmup plan for my fitness brand"
"Generate warmup content for my new Instagram account in the food niche"
"Help me warm up my Instagram before posting promotional content"
"Build an Instagram engagement plan for Days 1 through 7"
"Create a warmup Reel brief for Instagram"
"Assess my Instagram warmup performance — should I start posting now?"
"I bought an Instagram account — how do I warm it up?"
"My Instagram reach is tanking — is this a trust score issue?"
```

## What It Does

- Generates a 14-day engagement plan split into five phases, customized to your niche
- Creates Days 8-10 first warmup content: Reel scripts and carousel briefs with image/video direction, caption, and hashtag strategy (5-hashtag hard limit enforced)
- Explains how Meta's trust score works, including the 2026 Stories weighting update
- Provides Stories warmup strategy with daily cadence and engagement sticker guidance
- Covers the signal hierarchy (shares > saves > comments > likes) and how to optimize for it
- Outputs a 48-hour post-warmup assessment framework with green/yellow/red signal thresholds
- Guides the graduated promotional content transition (Weeks 3-6 cadence)
- Documents Instagram suppression triggers specific to new accounts and automation patterns

## Phases Covered

| Phase | Days | Output |
|-------|------|--------|
| Pure Consumption | 1-3 | FYP behavior, Explore seeding, no engagement yet |
| Light Engagement | 4-7 | Follow criteria, comment templates, Stories interactions, daily checklist |
| First Post | 8-10 | Reel or carousel warmup brief, caption, hashtags, Stories announcement |
| Assessment | Day 12 (48h check) | Performance signals, go/no-go framework |
| Gradual Scaling | 11-14 | Second post decision, Stories cadence, content pillar introduction |
