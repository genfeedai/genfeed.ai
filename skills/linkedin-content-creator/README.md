# LinkedIn Content Creator

Expert LinkedIn content strategist for professional growth. Creates posts, articles, carousels, and newsletters optimized for the LinkedIn algorithm and professional engagement.

## Installation

```bash
npx skills add genfeedai/skills/linkedin-content-creator
```

## Usage

```
"Write a LinkedIn post about my career transition"
"Create a thought leadership article on remote work trends"
"Draft a LinkedIn carousel on leadership lessons"
"Write a newsletter edition on AI in enterprise"
"Help me craft a contrarian post about hustle culture"
```

## What It Does

- Writes posts in 5 proven formats: story, insight, contrarian, listicle, celebration
- Structures long-form articles (1500-2500 words) with SEO-friendly headings and pull quotes
- Creates carousel/PDF outlines with cover slides, value slides, and CTAs
- Drafts newsletter editions with subscriber growth hooks
- Applies hook patterns proven to drive dwell time and comments
- Optimizes for algorithm signals: dwell time, early comments, native uploads, hashtag limits
- Calibrates tone: authoritative but approachable, data-backed, genuinely vulnerable
