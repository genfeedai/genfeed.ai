# Newsletter Creator

Expert newsletter editor and email content strategist. Creates complete newsletter editions with A/B-tested subject lines, optimized preview text, editorial structures, and subscriber growth strategies across all major platforms.

## Installation

```bash
npx skills add genfeedai/skills/newsletter-creator
```

## Usage

```
"Write a weekly newsletter about AI trends in marketing"
"Create a newsletter edition announcing our product launch"
"Draft subject lines for my SaaS newsletter"
"Help me structure a curator-style newsletter"
"Write a newsletter with a deep-dive essay on remote work"
```

## What It Does

- Generates 5+ A/B subject line variants using proven formulas: curiosity, benefit, urgency, personal, question, number
- Optimizes preview text to complement (never repeat) the subject line
- Structures content in 5 editorial formats: Curator, Essayist, Reporter, Digest, Hybrid
- Places CTAs strategically with placement reasoning for each position
- Provides frequency and cadence guidance matched to content type
- Includes subscriber growth tactics: referral programs, lead magnets, cross-promotion
- Tracks key metrics with benchmarks: open rate, CTR, reply rate, growth rate
- Adapts to platform specifics: Beehiiv, Ghost, Substack, ConvertKit, Mailchimp, Buttondown
