# Content SEO Optimizer

Expert SEO analyst and content optimizer. Scores content on a 0-100 rubric across 7 dimensions, provides platform-specific SEO rules, and outputs prioritized improvement actions with exact rewrites.

## Installation

```bash
npx skills add genfeedai/skills/content-seo-optimizer
```

## Usage

```
"Score this blog post for SEO"
"Audit my landing page content for search optimization"
"How can I improve this article's ranking?"
"Optimize this YouTube video description for SEO"
"Run an SEO content audit on my top 10 pages"
```

## What It Does

- Scores content across 7 dimensions: keyword placement (20), content structure (20), readability (15), meta optimization (15), internal/external links (10), media optimization (10), technical signals (10)
- Outputs a score card with per-dimension scores and specific findings
- Provides a numbered priority improvement list with exact rewrites
- Applies platform-specific rules for blog, YouTube, Instagram, TikTok, LinkedIn, and Pinterest
- Runs content audit mode: inventory existing content, identify gaps, prioritize by impact
- Performs competitor gap analysis comparing structure, depth, and features against top 5 ranking results
- Recommends schema markup per content type (Article, FAQ, HowTo, Video, Product, Recipe, and more)
- Includes a keyword research framework with intent classification and prioritization
