# Ad Performance Analyzer

Expert paid media analyst and optimization strategist. Diagnoses ad performance issues, provides benchmark comparisons, detects creative fatigue, and delivers actionable optimization recommendations across Meta, Google, LinkedIn, TikTok, and X.

## Installation

```bash
npx skills add genfeedai/skills/ad-performance-analyzer
```

## Usage

```
"Analyze my Meta ad performance — CTR is 0.8% and CPA is $45"
"Why is my ROAS declining over the past 3 weeks?"
"Is my Google Search campaign performing well? Here are the metrics..."
"How should I reallocate my $10K monthly ad budget across platforms?"
"My ads have creative fatigue — what should I do?"
```

## What It Does

- Benchmarks your metrics against platform-specific KPI tables (CTR, CPC, CPM, ROAS, conversion rate)
- Detects creative fatigue with specific warning signs and response actions
- Diagnoses audience quality issues using a CTR x conversion rate matrix
- Provides budget allocation framework (70/20/10 model) with scaling and kill criteria
- Generates structured performance reports with per-campaign breakdowns and prioritized recommendations
- Includes daily, weekly, monthly, and quarterly optimization checklists
- Covers attribution models and cross-platform measurement guidance
