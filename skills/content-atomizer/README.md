# Content Atomizer

Expert content repurposing strategist. Takes a single piece of source content and systematically produces dozens of platform-optimized derivatives with adapted tone, format, length, and a staggered scheduling cadence.

## Installation

```bash
npx skills add genfeedai/skills/content-atomizer
```

## Usage

```
"Atomize this blog post into social content for all platforms"
"Repurpose this YouTube video into tweets, LinkedIn posts, and an IG carousel"
"Turn this podcast episode into a week of content"
"Create 10 derivatives from this newsletter essay"
"Break this report into shareable social pieces"
```

## What It Does

- Analyzes source content (blog, video, podcast, report, presentation) and extracts key insights, data points, quotes, stories, and frameworks
- Maps derivatives per platform: 1 blog post yields 15+ derivatives across X, LinkedIn, Instagram, TikTok, YouTube, newsletter, and Pinterest
- Adapts tone (casual for X, professional for LinkedIn, visual-first for IG), length (280 chars to 2500 words), and format (text, carousel, video script, audiogram brief)
- Creates a staggered 5-7 day scheduling cadence with platform-specific peak times
- Supports content recycling: evergreen content re-atomized quarterly with fresh angles, updated data, audience shifts, or seasonal hooks
- Outputs a complete atomization map with derivative table, full content pieces, and scheduling dates
