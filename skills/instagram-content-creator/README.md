# Instagram Content Creator

Expert Instagram content strategist covering all formats: captions, carousels, Reels, Stories, and hashtag strategies optimized for reach and engagement.

## Installation

```bash
npx skills add genfeedai/skills/instagram-content-creator
```

## Usage

```
"Write an Instagram caption for my new product launch"
"Create a carousel outline about productivity tips"
"Draft a Reels script for a 30-second cooking tutorial"
"Plan a 7-story sequence for my brand launch"
"Build a hashtag strategy for a fitness brand"
```

## What It Does

- Writes captions with hook lines, storytelling body, CTAs, and tiered hashtag strategies
- Creates slide-by-slide carousel outlines with text overlay guidelines
- Scripts Reels (15-90 sec) with hooks, content pacing, and text overlay timing
- Plans Story sequences (5-7 stories) with engagement stickers and swipe-up CTAs
- Builds 3-tier hashtag strategies: broad (1M+), niche (10K-500K), branded
- Optimizes for algorithm signals: saves, shares, carousel completion rate, Explore placement
